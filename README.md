# Audio--text
Название: конвертер видео/аудиофайлов в текст.
Описание: приложение предназначено для конвертации аудио/видеофайлов в текст или текстовый файл. Затем пользователь может скачать файл или просто скопировать текст.
Целевая аудитория: блогеры, видеомейкеры, копирайтеры.
Ключевой функционал: загрузка видео/аудиофайлов, конвертация видео/аудиофайлов в текст или текстовый файл, скачивание текстового файла или копирование текста напрямую.

Структура приложения:
Главная (/) - публичная страница, позволяет пользователю использовать основные функции приложения.
Аутентификация (/auth) - вход, регистрация, восстановление пароля.
Поддержка (/help) - связь с разработчиком.
Поддержать разработчика(/donate) - позволяет поддержать разработчика материально.
Взаимодействие с API (/api) - получение ключа API.
Руководство по API (/api_manual) - содержит руководство по использованию API.
Оплата использования API (/pay) - позволяет пользователю оплатить использование API.
О разработчике (/about) - позволяет пользователю ознакомиться с другими проектами разработчика.

Технологический стек: 
Frontend: Vue.js, Vue Router 4, Pinia.
Backend: Python, Flask, SQLite.
Инфраструктура: frontend на Vercel, backend и БД локально.

Модели данных:
Сущность: User - основные поля: ID, username, email, password_hash, show_add, api_tier.
Сущность: API_tier - основные поля: id, name, token_limit.
Сущность: User_Tokens - основные поля: user_id, balance, last_replenishment_at.

Ключевые эндпоинты API:
Регистрация: 
Метод: POST.
Путь: /auth/register.
Описание: создает новый аккаунт пользователя.
Body(JSON): {"email":"..." , "password":"...", "username":"..."}.
Ответ: 201 Created и данные созданного пользователя(без пароля).

Вход:
Метод: POST.
Путь: /auth/login
Описание: проверяет учетные данные и разрешает доступ.
Body(JSON): {"email":"..." , "password":"..."}
Ответ: 200 OK и id пользвателя.

Получение количества оставшихся токенов:
Метод: GET.
Путь: /api/token.
Описание: возвращает количество осташихся токенов.
Body(JSON): {"id":"..."}
Ответ: 200 OK и количество токенов.
